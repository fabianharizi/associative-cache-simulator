Direct-Mapped Cache Simulator

How to Use

  1. Enter a memory address in the input field below.
  2. Click the "Access Memory" button.
  3. The table shows each cache line, its tag, and stored data.
  4. Cache lines highlight in green for hits and red for misses.
  5. The memory address is also displayed in binary for clarity.
